# 🏠 Prekaria

A personal toolchain for collecting, tracking, and analyzing housing market data across multiple platforms.

## 📖 Overview

Prekaria helps you monitor real estate listings from different housing platforms in one place. It enables you to compare properties side-by-side, track price history over time, and build a personal dataset of the housing market in your area of interest.

## ✨ Features

- 🌐 **Multi-platform collection** — Scrape and aggregate listings from multiple housing platforms
- 📈 **Price history tracking** — Record and visualize how listing prices change over time
- ⚖️ **Property comparison** — Compare listings across platforms using normalized data
- 💾 **Local data storage** — Keep a personal historic dataset of housing market snapshots
- 🔍 **Analysis tools** — Query and analyze collected data to identify trends and anomalies

## 🎯 Use Case

This tool is meant for personal use by anyone wanting to:

- 📊 Stay informed about price trends in a specific area or neighborhood
- 🔔 Detect when a listing drops in price across platforms
- 📌 Avoid losing track of interesting properties over time
- 🗂️ Build a historic record of market conditions for future reference
